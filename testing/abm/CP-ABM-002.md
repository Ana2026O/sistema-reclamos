# Caso de Prueba: CP-ABM-003

## Módulo
ABM Categorías

## Descripción
Validar que el sistema permita modificar una categoría existente.

## Precondiciones
- Administrador u Operador autenticado en el sistema.
- Categoría existente en la base de datos.

## Pasos
1. Acceder al listado de categorías.
2. Seleccionar una categoría existente.
3. Modificar los datos requeridos.
4. Presionar el botón **Guardar cambios**.

## Resultado esperado
La categoría se actualiza correctamente y se muestra en el listado.

## Evidencia
Captura de pantalla de la categoría modificada en el select del bton de gestion de categorias
