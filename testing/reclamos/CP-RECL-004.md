# Caso de Prueba: CP-RECL-004

## Módulo
Reclamos

## Descripción
Validar que el sistema genere automáticamente un número de reclamo al enviar el formulario.

## Precondiciones

- Datos de reclamo válidos disponibles.

## Pasos
1. Acceder al formulario de alta de reclamo.
2. Completar los campos obligatorios con datos válidos.
3. Presionar el botón **Enviar reclamo**.

## Resultado esperado
El sistema genera un número de reclamo único y lo muestra en el detalle.

## Evidencia
Captura de pantalla del número de reclamo generado.
