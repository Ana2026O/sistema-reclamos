# Caso de Prueba: CP-RECL-002

## Módulo
Reclamos

## Descripción
Validar que el sistema permita modificar los estado y proridad de un reclamo existente.

## Precondiciones
- Usuario autenticado en el sistema.
- Reclamo existente en la base de datos.

## Pasos
1. Acceder al listado de reclamos.
2. Seleccionar un reclamo existente.
3. Modificar los datos requeridos.
4. Presionar el botón **Guardar cambios**.

## Resultado esperado
El reclamo se actualiza correctamente y se muestra en el listado y detalle.

## Evidencia
Captura de pantalla del reclamo actualizado con acmbio en priorida y estado.
