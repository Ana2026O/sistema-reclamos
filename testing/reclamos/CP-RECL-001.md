# Caso de Prueba: CP-RECL-001

## Módulo
Reclamos

## Descripción
Validar que el sistema permita dar de alta un reclamo con datos válidos.

## Precondiciones

- Datos de reclamo válidos disponibles.

## Pasos
1. Acceder al formulario de alta de reclamo.
2. Completar los campos obligatorios con datos válidos.
3. Presionar el botón **Enviar reclamo**.

## Resultado esperado
El reclamo se guarda correctamente y aparece en la lista de reclamos.

## Evidencia
Captura de pantalla del reclamo listado.
