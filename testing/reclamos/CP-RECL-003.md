# Caso de Prueba: CP-RECL-003

## Módulo
Reclamos

## Descripción
Validar que el sistema permita consultar un reclamo y mostrar información de estados.

## Precondiciones
- Usuario autenticado en el sistema 
- Numero de referencia generado
- Reclamo existente en la base de datos.

## Pasos
1. Acceder al módulo de consulta de reclamos.
2. Ingresar el número de reclamo.
3. Presionar el botón **Consultar**.

## Resultado esperado
El sistema muestra la información de estados del reclamo con cartel

## Evidencia
Captura de pantalla de la consulta realizada.
